GridtoFlex

Flexbox fallbacks for popular UI solutions that use CSS Grid.

CSS grid is AMAZING! However, if you need to support users of IE11 and below, or Edge 15 and below, grid won't really work as you expect (more info here). This site is a solution for you so you can start to progressively enhance without fear!

Things to Note: CSS grid allows for specified styling using the vertical area as well as horizontal. Flexbox doesn't allow for this, so we'll be using fixed heights in these examples. Also, because of how much power grid gives us, some of these are not the most elegant markup you could use when writing with grid-only. They are instead all solutions that allow for a flexbox fallback without changing markup, so you can progressively enhance your user's experience.

The components would also take the height of the internal elements. Feel free to play around with the CSS. To swap between implementations, use the toggles. You can also play with these in Codepen (which may have better syntax highlighting!).

The code blocks are fully editable, and you can update the shared variables here, too (but note that browsers which don't support grid also don't support variables, so use this only for understanding, and absolute values in prod!)
